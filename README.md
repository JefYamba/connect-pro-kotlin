# Connect PRO

Connect Pro est une plateforme de mise en relation professionnelle (portfolio, marketplace, recommandations, etc.).  
**Backend migré de Java vers Kotlin + Spring Boot.**

## Technologies

- **Langage** : Kotlin 2.x
- **Framework** : Spring Boot 4.0.5
- **Base de données** : PostgreSQL + JPA + Flyway
- **Authentification** : Spring Security + JWT
- **Autres** : AWS S3, WebSocket, Resend (email), OpenAPI (Swagger)
